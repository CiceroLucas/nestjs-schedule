
## Description

Esse projeto foi feito para aplicar meus conhecimentos em agendamento de email usando o Schedule do NestJS

## Technologies
* NestJS
* PostgreSQL
* TypeORM

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
